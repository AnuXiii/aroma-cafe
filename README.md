# Project Dependencies

### General Information

- **Name** : Aroma Cafe
- **Theme**: Dark - Gold - Orange - Cream
- **Icons**: ionicons
- **Fonts**: Dana & Poppins

### Tools & Frameworks

- **CSS Framework**: TailwindCSS
- **Package Manager/Project Builder**: Vite

### Features & Libraries

- **Alerts**: Toastify.js
- **Animations**: Gsap.js - ScrollTrigger
- **Contacting**: Email.js
- **Slider**: Swiper.js
- **Maps**: Neshan
